# Архив Lunaria Fox

Внутри архива две основные папки:

- `бот дискорд` — исходники и конфигурация Discord-бота;
- `сайт бот` — сайт, дашборд и конфигурация под Vercel.
- в документации лежит и SQL для premium-слоя, и SQL для control-plane sync.

Рекомендуемый порядок:

1. Сначала настроить и проверить `бот дискорд`.
2. Затем настроить `сайт бот`.
3. После этого подключить одинаковые значения Supabase и Discord OAuth.
