Положи сюда арты для уведомлений о выбросах.

Рекомендуемые имена:
- emission-01.png
- emission-02.png
- emission-03.png
- emission-04.png
- emission-05.png
- emission-06.png

После загрузки файлов можно указать их в EMISSION_ART_URLS, например:
https://your-site.example/emissions/emission-01.png,https://your-site.example/emissions/emission-02.png

Важно:
- картинки из чата не попадают сюда автоматически;
- бот может использовать только реальные файлы в проекте или прямые URL.
